Introduction to the Hex-Rays decompiler

IDA Pro is PE disassembler application. A disassembler is used to translate machine code into a readable format called assembly language. IDA Pro can also also include the Hex-rays decompiler which can translate the binary code into pseudocode of each function making it more understandable

The advantages of this decompiler is it requires less time to read the the code, the program logic is more obvious and you don't have to learn assembly language. The code is generated on the fly making it fast enough to analyze 99% of functions within a couple of seconds.

The decompiler supports the x86, x64, ARM32, ARM6, PowerPC processors and runs on Windows, Linux and Mac OS X.
